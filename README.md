# NextGit

This is a work in progress, nothing here works yet!

But, when it does:
NextGit is a Glulx interpreter for the ZX Spectrum Next based on Git, with a native NextGlk implementation, SD card support, Layer 2 graphics, and optional Pi acceleration.

The plan is to take the current version of Git, implement our own Glx that just handles basic I/O to get a text adventure working minus graphics and sound, get that running on Linux, and then to port the result to the Next.  Initial viability investigations look promising.
